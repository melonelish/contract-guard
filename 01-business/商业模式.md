## 一、商业模式画布

| 维度 | 内容 |
|---|---|
| **客户细分** | ①小微企业（无自有法务）②中型企业（有1-2名法务）③律所/大型企业法务部 |
| **价值主张** | 3分钟看懂合同风险，每条风险都有法条依据和修改建议 |
| **渠道** | 官网直销、微信小程序、合作伙伴（代理记账/产业园/律所）、内容营销（知乎/公众号） |
| **客户关系** | 自助式 SaaS + 企业版专属客户成功经理 |
| **收入来源** | SaaS 订阅费 + 私有化部署费 + API 调用费 |
| **核心资源** | LLM Agent 架构、中国法律知识库、合同审查规则库 |
| **关键业务** | 产品迭代、知识库维护、市场推广、客户服务 |
| **重要伙伴** | LLM 服务商、律师/法务专家、渠道合作伙伴 |
| **成本结构** | LLM API 费用(40%)、人力(35%)、服务器(10%)、市场(15%) |

## 三、客户生命周期

### 3.1 用户旅程

```
认知 ──→ 试用 ──→ 付费 ──→ 复购 ──→ 推荐
  │        │        │        │        │
 内容    免费1次    按次     买套餐    口碑
 营销    体验       购买     企业版    裂变
```

### 3.2 核心漏斗指标（目标）

| 环节 | 转化率 |
|---|---|
| 访问 → 注册 | 8% |
| 注册 → 使用免费审查 | 50% |
| 免费试用 → 首次购买 | 8% |
| 首次购买 → 二次购买 | 35% |
| 二次购买 → 购买套餐 | 15% |
| NPS | > 40 |
