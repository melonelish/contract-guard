## I. Business Model Canvas

| Dimension | Content |
|---|---|
| **Customer Segments** | ① Micro/Small Enterprise (no in-house legal) ② Medium Enterprise (1-2 in-house legal staff) ③ Law Firms/Large Enterprise Legal Departments |
| **Value Proposition** | Understand contract risks in 3 minutes, each risk comes with statute citations and modification suggestions |
| **Channels** | Direct sales via website, WeChat Mini Program, partners (bookkeeping agencies/industrial parks/law firms), content marketing (Zhihu/Official Accounts) |
| **Customer Relationships** | Self-service SaaS + dedicated Customer Success Manager for enterprise edition |
| **Revenue Streams** | SaaS subscription fees + private deployment fees + API usage fees |
| **Key Resources** | LLM Agent architecture, Chinese law knowledge base, contract review rule base |
| **Key Activities** | Product iteration, knowledge base maintenance, marketing, customer service |
| **Key Partners** | LLM service providers, lawyers/legal experts, channel partners |
| **Cost Structure** | LLM API costs (40%), HR (35%), servers (10%), marketing (15%) |

## II. Pricing Strategy

### 2.1 Product Matrix (Per-Use Pricing)

| Plan | Price | Review Count | Description |
|---|---|---|---|
| **Free Trial** | ¥0 | 1 use | Sign up and get, includes 3 rounds of review-edit-re-review loop |
| **Single Purchase** | ¥19 | 1 use | Same contract, 3 free re-reviews within 7 days |
| **5-Use Pack** | ¥79 | 5 uses | ¥15.8/use, save 17% |
| **20-Use Pack** | ¥249 | 20 uses | ¥12.45/use, save 34% |
| **Enterprise Annual - Basic** | ¥2,999/year | 200 uses | Includes 3 team member seats |
| **Enterprise Annual - Premium** | ¥9,999/year | 1,000 uses | Unlimited seats + API access |
| **Private Deployment** | ¥200,000+ | On demand | One-time license + 20% annual maintenance fee |

> **See 《Pricing & Cost Risk Control.md》 for detailed pricing strategy**

### 2.2 Marginal Cost

| Cost Item | Per Contract (15 pages) | Per Contract (30 pages) |
|---|---|---|
| LLM Token Consumption | ~¥1.0 | ~¥2.5 |
| Document Parsing | ~¥0.2 | ~¥0.3 |
| Vector Retrieval | ~¥0.1 | ~¥0.1 |
| Storage & Bandwidth | ~¥0.05 | ~¥0.05 |
| **Total Marginal Cost per Contract** | **~¥1.35** | **~¥2.95** |

Per-use price ¥19, standard contract (15 pages) marginal cost ¥1.35, **gross margin 93%**. Large contract (30 pages) marginal cost ¥2.95, **gross margin 84%**.
