## 三、关键假设

| 假设 | 数值 | 依据 |
|---|---|---|
| 月自然增长 | 20-30% | 内容营销 + SEO |
| 免费到付费转化 | 1%→6%（18个月内逐步提升） | 产品成熟度 + 付费引导优化 |
| 月流失率 | 5-8% | SaaS 行业基准 |
| LLM API 成本 | 年降 15-20% | 模型降价趋势 |
| 单用户获取成本（CAC） | ¥50-80 | 内容营销为主，付费投放为辅 |
| 毛利率 | 75-85% | 软件服务行业基准 |
