# 产品定位

> 版本：v1.0 | 最后更新：2026-06-15

---

## 一、一句话定位

**"3 分钟看懂一份合同的风险"** — 面向中国 4000 万中小企业的一键合同风险扫描工具。

---

## 二、品牌故事

中国有 4000 万家中小企业，90% 没有法务。每签一份合同都是赌博——你不知道第 7 页的违约金条款是不是合法、第 12 页的管辖约定是不是坑你、合同里是不是少了保密条款。

ContractGuard 做的事情很简单：你把合同丢进来，它帮你把每一条风险标出来，告诉你哪条违法了、哪条不公平、应该改成什么样。不是替代律师，而是让你在见到律师之前就知道问题在哪。

---

## 三、核心价值主张

| 价值维度 | 描述 |
|---|---|
| **省时** | 人工审查 3 小时 → 系统 15 分钟 |
| **省钱** | 律师审查 300 元/份 → 系统 3 元/份 |
| **省心** | 不遗漏，每条风险都标注原文引用 |
| **可信** | 每条结论关联法条原文 + 相似判例 |

## 四、用户心智模型

### 使用前

> "这份合同有没有问题？我看不懂，但签字了就要负责。要不花 500 块找个律师看看？又不知道靠不靠谱..."

### 使用后

> "上传→等 15 分钟→看到 3 个红色风险、5 个黄色提示。违约金太高了，修改建议说压到 20%。我直接把报告发给对方，改完再签。踏实。"

## 六、产品边界（不是什么）

| 边界 | 说明 |
|---|---|
| **不是律师** | 不提供法律意见，不代写合同，不出庭 |
| **不是电子签章** | 不替代法大大、上上签的签章功能 |
| **不审判对方** | 只分析风险，不判断"对方是不是骗子" |
| **非合同不审** | 不审起诉书、判决书、行政文书 |
| **不保 100% 准确** | 定位为辅助工具，重要合同仍需律师确认 |

## 七、差异化定位矩阵

```
                审查深度 ↑
                         │
            ContractGuard ●  （LLM语义理解+多步推理+法条引用）
                         │
                         │
       幂律智能、华宇元典 ●
                         │
  ──────────────────────────────→ 审查广度
     LawGeex ●         │
     (英文合同)        │   法大大、上上签 ●
                       │   （模板匹配+签章闭环）
                       │
                       │
                审查深度 ↓
```

ContractGuard 的定位：**最深的审查深度 × 最广的中国法律覆盖**。

## 八、品牌调性

| 维度 | 风格 |
|---|---|
| **理性** | 每一条结论都有法条/判例支撑，不做无依据的判断 |
| **亲民** | 把"违约金不得超过造成损失的百分之三十"翻译成"你写50%，法院最多判30%" |
| **可靠** | 免责声明清晰、置信度标注明确、不确定的就说不知道 |
