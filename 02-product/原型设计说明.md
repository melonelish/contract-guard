> Version: v1.0 | Last Updated: 2026-06-15

---

## 1. Core Page Inventory

| Page | Route | User Actions |
|---|---|---|
| Home / Landing Page | `/` | Product introduction, CTA to guide registration |
| Login / Register | `/login` `/register` | Email / WeChat login |
| Contract Upload | `/upload` | Drag-and-drop upload PDF/Word, view upload progress |
| Review Report (Single Pane) | `/review/:id` | Left: original contract + Right: risk list, review suggestions item by item |
| Built-in Editor | `/review/:id/edit` | Online contract editing, AI suggestions embedded, one-click apply |
| Print Modal | Modal | Select version, paper format, preview & print |
| Contract List | `/contracts` | View / search / filter historical contracts |
| Review History | `/reviews` | View all past review records |
| Account Management | `/account` | View balance, purchase credits, API configuration (Enterprise) |

---

## 2. Core Interaction Flow

See **"Dual-Pane Comparison and Diff Visualization Design.md" Chapter 9: Complete Closed-Loop Flow** for details.

Summary:
1. Upload contract → AI review → Left: original text + Right: risk list
2. Enter edit mode → Risk annotations embedded in editor → One-click apply / manual edit
3. Re-review → Incremental analysis of changed clauses
4. Satisfied → Export / Print (three versions available)

---

## 3. Design Principles

- **WYSIWYG**: Review suggestions are embedded directly in the contract text — no need to cross-reference two documents
- **Keep Users In-System**: Edit, review, and print all completed online without downloading locally
- **Mobile Adaptation**: WeChat Mini Program primarily supports upload and viewing; editing is done on PC
