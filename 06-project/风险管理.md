## 2. Risk Mitigation Measures

### R01 — LLM API Instability

```python
# Multi-model hot-switching
MODEL_FALLBACK_CHAIN = [
    "mimo2.5",              # Primary
    "deepseek-v4-flash",     # Backup
]
```

### R05 — Legal Compliance Risk

Six-layer guardrails:
1. Analyzer Agent Prompt hard constraint (must not provide legal advice)
2. Validator Agent content filtering
3. Mandatory disclaimer at end of report
4. User agreement clearly stating "assistive tool" positioning
5. No legal qualification warning
6. Legal consultant periodic spot-checks
