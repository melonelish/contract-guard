## 二、RAG 检索工具

### 2.1 接口定义

```yaml
tool: rag_search
description: 在知识库中检索相关内容
parameters:
  query:
    type: string
    description: 检索查询文本
    required: true
  knowledge_base:
    type: enum
    enum: ["laws", "cases", "contract_templates", "review_rules"]
    description: 目标知识库
    required: true
  strategy:
    type: enum
    enum: ["semantic", "keyword", "hybrid"]
    default: "hybrid"
    description: 检索策略
  top_k:
    type: integer
    default: 5
    minimum: 1
    maximum: 20
    description: 返回结果数量
  filters:
    type: object
    description: 多维过滤条件
    properties:
      law_type:
        type: string
        description: 法律来源过滤（如"民法典"）
      court_level:
        type: array
        description: 法院级别过滤
      year_range:
        type: array
        items: integer
        description: 年份范围过滤
  output_format:
    type: enum
    enum: ["full", "summary"]
    default: "full"
    description: 返回详细还是摘要
```

### 2.2 调用示例

```python
# Python SDK 调用
result = rag_search(
    query="违约金不得超过造成损失的百分之三十",
    knowledge_base="laws",
    strategy="hybrid",
    top_k=5,
    filters={"law_type": "民法典"},
    output_format="full"
)
```

## 三、LLM 调用工具

### 3.1 标准调用

```python
# 标准 LLM 调用（通过统一封装层）
response = llm_chat(
    model="mimo2.5",  # 或 "deepseek-v4-flash"
    system_prompt=AGENT_SYSTEM_PROMPTS["analyzer"],
    user_message=build_analyzer_message(clause_json),
    temperature=0.1,  # 法律分析需要低温度（确定性高）
    max_tokens=2000,
    response_format={"type": "json_object"}  # 强制 JSON 输出
)
```

### 3.2 模型选择策略

| 任务 | 模型 | 理由 |
|---|---|---|
| 文档解析（非表格） | DeepSeek V4-Flash | 成本极低，文本处理够用 |
| 文档解析（含复杂表格） | MiMo 2.5 | 表格理解能力强 |
| 法律条款分析 | MiMo 2.5 | 推理能力要求高 |
| 报告生成 | DeepSeek V4-Flash | 结构化文本生成，成本低 |
| 报告校验 | MiMo 2.5 | 质量检查不容出错 |

### 3.3 Token 消耗预估

```
单份 30 页采购合同审查的 Token 消耗估算：

  Parser:     3,000 tokens      × ¥0.005/1K = ¥0.015
  Analyzer:   40 条条款 × 1,200 = 48,000 tokens × ¥0.02/1K = ¥0.96
  Report:     5,000 tokens      × ¥0.005/1K = ¥0.025
  Validator:  2,000 tokens      × ¥0.02/1K  = ¥0.04
  ─────────────────────────────────────────────────
  合计：约 ¥1.04 / 份（API 成本仅 LLM 部分）
  加上 Embedding + 向量检索 + 存储：约 ¥2.35 / 份
```

## 四、容错与降级

```python
# LLM 调用自带重试与降级
def llm_chat_with_fallback(**kwargs):
    primary_model = kwargs.get("model", "mimo2.5")
    fallback_models = ["deepseek-v4-flash"]
    
    models_to_try = [primary_model] + fallback_models
    
    for model in models_to_try:
        try:
            kwargs["model"] = model
            result = _call_llm_api(**kwargs)
            return result
        except RateLimitError:
            await asyncio.sleep(2)  # async sleep, 不阻塞事件循环
            continue
        except ServiceUnavailableError:
            continue
        except TimeoutError:
            continue
    
    raise AllLLMsUnavailable("所有 LLM 模型均不可用")
```
