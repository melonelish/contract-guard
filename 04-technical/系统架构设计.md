## 2. Technology Decisions

| Layer | Technology | Rationale |
|---|---|---|
| **Frontend** | React 18 + TypeScript + Ant Design | Rich component ecosystem; WeChat Mini Program via Taro cross-platform |
| **API Gateway** | Kong / Nginx | Production-grade gateway with rate-limiting and auth plugins |
| **Backend Framework** | Python FastAPI | Async high-performance, best AI ecosystem (LangChain/LlamaIndex are native Python) |
| **Agent Framework** | Self-built (no LangChain dependency) | Precise control over Agent communication and retry logic |
| **LLM SDK** | OpenAI-compatible + multi-model adapter | Supports hot-switching between MiMo 2.5 / DeepSeek V4-Flash |
| **Message Queue** | Redis Streams | Lightweight; no need for separate RabbitMQ/Kafka deployment |
| **Database** | PostgreSQL 15 | Mature, stable; supports JSONB and pgvector extension |
| **Vector DB** | Milvus (prod) / pgvector (dev) | Multi-dimensional filtering + hybrid search |
| **File Storage** | MinIO (S3-compatible) | On-premise friendly, S3 API compatible |
| **Keyword Search** | Elasticsearch 8.x | Chinese tokenization + BM25 retrieval |
| **Deployment** | Docker Compose (dev) → K8s (prod) | Progressive architecture |
| **Observability** | Prometheus + Grafana + LangSmith | Full-stack observability |

## 4. Security Architecture

### 4.1 Defense-in-Depth

```
  Network     TLS 1.3 end-to-end encryption
     │
  Application  JWT auth + RBAC permissions
     │
  Data        AES-256 encryption at rest
     │         Contract text encrypted; PII stored separately
     │
  Audit       Full operation log; immutable; non-deletable
```

### 4.2 Multi-Tenant Isolation

```
           ┌────────────┐
           │   Shared    │ ← Legal knowledge base (all tenants, read-only)
           │   Layer     │
           └─────┬──────┘
                 │
     ┌───────────┼───────────┐
     │           │           │
┌────▼────┐ ┌────▼────┐ ┌────▼────┐
│ Tenant  │ │ Tenant  │ │ Tenant  │
│    A    │ │    B    │ │    C    │
│         │ │         │ │         │
│ Contracts│ │Contracts│ │Contracts│
│ History  │ │ History │ │ History │
│ Custom   │ │ Custom  │ │ Custom  │
│ Rules    │ │ Rules   │ │ Rules   │
└─────────┘ └─────────┘ └─────────┘
```

SQL queries auto-inject `WHERE tenant_id = {current_tenant}`; database row-level security as safety net.
