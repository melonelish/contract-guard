1. **React Hook 规则违反**：`useMemo` 在条件 `return` 后调用 → 已修复，所有 hooks 移到条件分支前
2. **请求失败卡死**：`fetchReport` throw 导致页面永远 Skeleton → 已修复，新增 `loadError` 状态和错误兜底
3. **未使用 import**：`Progress` 组件不再使用 → 已清理
4. **中文错误消息**：axios 错误消息改为中文友好（404/500/通用）

### 降级链路验证方式

开发态可通过以下方式验证"WS 失败后自动降级轮询"：

1. **强制轮询开关**（推荐）：在浏览器控制台执行 `localStorage.setItem("force_polling", "true")`，刷新页面后触发审查，页面将直接进入轮询模式，不尝试建立 WS。
2. **环境变量**：启动前端时设置 `VITE_FORCE_POLLING=true`，所有审查均走轮询。
3. **关闭后恢复**：执行 `localStorage.removeItem("force_polling")` 并刷新页面，恢复 WS 优先模式。
