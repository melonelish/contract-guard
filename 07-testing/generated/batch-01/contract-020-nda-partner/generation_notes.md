# 生成说明 - contract-020

## 合同背景

本合同为合作伙伴保密协议，背景是某精密制造企业（宏远精密制造）与特种材料研发企业（瑞联新材料）就新一代航空航天复合材料联合研发项目签订的保密协议。双方在合作过程中需要交换各自的核心技术秘密、工艺参数和商业信息。

这份合同的业务场景具有以下特点：
1. 涉及航空航天领域的核心技术，保密价值极高
2. 联合研发项目周期较长（3年），保密期限需要合理设定
3. 涉及背景知识产权和前景知识产权的归属问题
4. 双方互为披露方和接收方，需要平衡双方的权利义务
5. 涉及跨境仲裁条款，需要考虑法律适用和仲裁地的协调

## 矛盾设计意图

### C001 - 保密期限与合作期限矛盾（1年 vs 3年）
**构造方式：** 合作期限约定为3年，但保密期限仅为1年，保密期限明显短于合作期限。
**设计目的：** 测试系统对保密期限与业务场景匹配性的检测能力。

### C002 - 适用法律与仲裁地冲突（中国法 vs 新加坡仲裁）
**构造方式：** 适用中国法律但仲裁地在新加坡，形成法律适用和仲裁程序的冲突。
**设计目的：** 测试系统对跨境争议解决条款协调性的检测能力。

## 测试价值说明

本合同样本对ContractGuard系统具有以下独特测试价值：

1. **联合研发项目特殊性检测：** 测试系统是否了解联合研发项目的特殊需求，如背景知识产权保护、成果归属、竞业限制等。

2. **保密期限与业务场景匹配性判断：** 测试系统是否能够发现保密期限与合作期限的不匹配问题。

3. **跨境争议解决条款审查：** 测试系统是否能够识别适用法律与仲裁地的冲突，以及仲裁语言与适用法律的不匹配。

4. **知识产权与保密义务交叉审查：** 测试系统是否能够识别保密信息中包含已有专利的问题。

5. **信息使用限制清晰度判断：** 测试系统是否能够识别模糊的信息使用限制条款。

6. **违约救济体系完整性检查：** 测试系统是否能够识别禁令救济等紧急救济手段的缺失。

本样本覆盖了合作伙伴保密协议审查的主要法律维度，特别是联合研发项目的特殊条款需求和跨境争议解决条款的协调性，是一个综合性较强的测试样本。
