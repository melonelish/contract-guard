## 风险类型覆盖

- 付款条件失衡
- 违约责任过重或过轻
- 知识产权归属不清
- 验收标准模糊
- 保密义务缺失
- 自动续约陷阱
- 管辖条款偏置
- 数据安全责任不清
- 交付范围不清
- 金额、日期、主体、附件引用前后冲突

## 使用方式

1. 将合同正文(contract_source.md)输入ContractGuard系统
2. 系统输出与expected_*.json进行对比
3. 计算风险识别准确率、漏检率、误检率
4. 验证矛盾检测和缺失条款检测能力
