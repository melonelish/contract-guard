# P1 样本：双向保密协议（S004）

## 概述

- **合同类型**：保密协议
- **样本 ID**：S004
- **优先级**：P1（高价值补充验证）
- **页数**：6 页
- **条款数**：10 条

## 测试重点

| 焦点 | 说明 |
|------|------|
| 保密期限 | 3.1条10年期限与3.2条"永久有效"存在逻辑冲突，检测矛盾分析能力 |
| 保密范围 | 定义过于宽泛（"包括但不限于"），是否识别对接收方的不公平条款 |
| 违约责任 | 50万元固定违约金是否合理，有无分级处罚机制 |

## 预期风险分布

| 等级 | 数量 | 关键风险 |
|------|------|----------|
| High | 1 | 保密期限逻辑矛盾（10年 vs 永久） |
| Medium | 2 | 保密范围过宽、违约金未分级 |
| Low | 1 | 管辖条款歧义（双向保密协议中"披露方所在地"指向不明） |

## 预期缺失条款

- **竞业限制/禁止招揽条款**（medium）：缺少对接触对方员工和客户资源的限制
- **审计权条款**（low）：无法验证接收方是否切实履行保密义务

## 已知矛盾

第三条 3.1（保密期限10年）与第三条 3.2（保密义务延续至信息进入公有领域）存在逻辑冲突。
