# P0 样本：普通员工劳动合同（S002）

## 概述

- **合同类型**：劳动合同
- **样本 ID**：S002
- **优先级**：P0（核心链路验证）
- **页数**：8 页
- **条款数**：22 条（含子条款）

## 测试重点

| 焦点 | 说明 |
|------|------|
| 竞业限制 | 覆盖全国、无补偿——典型案例，检验 AI 是否能识别违法竞业限制 |
| 试用期 | 6个月常规试用期 + 可延长至12个月，严重违法，检验 AI 法条引用准确性 |
| 解除条件 | 单方调岗调薪权 + 解除条件宽泛，检验 AI 对劳动者保护条款的敏感度 |

## 预期风险分布

| 等级 | 数量 | 关键风险 |
|------|------|----------|
| High | 4 | 加班费打包违法、无补偿竞业限制、试用期超法定上限、自愿放弃社保违法 |
| Medium | 3 | 单方调薪权过宽、单方调岗权缺限制、社保低基数不合规 |
| Low | 2 | 加班审批举证不对等、争议管辖地不利 |

## 预期缺失条款

- **合规社保缴纳条款**（high）：现有社保条款存在按最低基数和允许弃保的违法约定
- **竞业限制补偿条款**（high）：有义务无补偿
- **带薪年休假具体安排**（low）：仅引用员工手册

## 已知矛盾

无内部条款矛盾。但多项条款与《劳动合同法》强制性规定存在外部冲突。
