# P0 样本：设备采购合同（S001）

## 概述

- **合同类型**：采购合同
- **样本 ID**：S001
- **优先级**：P0（核心链路验证）
- **页数**：12 页
- **条款数**：18 条（含附件）

## 测试重点

| 焦点 | 说明 |
|------|------|
| 预付款比例 | 30%预付款是否偏高，付款节奏对甲方的保护是否充分 |
| 违约金上限 | 两条违约金条款(3.4 vs 10.2)比例不一致的内部冲突检测 |
| 验收标准 | 3次验收不合格才解约是否合理，违约金能否覆盖损失 |

## 预期风险分布

| 等级 | 数量 | 关键风险 |
|------|------|----------|
| High | 3 | 预付款比例过高、违约金冲突、验收处罚偏轻 |
| Medium | 4 | 交付起算模糊、质保起算不利、通用违约金偏低、管辖条款效力 |
| Low | 2 | 保密期限偏短、售后响应无违约后果 |

## 预期缺失条款

- **质量异议处理条款**（medium）：缺少异议期限和处理流程
- **培训与交付文档条款**（low）：未明确培训、操作手册等交付物

## 已知矛盾

第三条 3.4（延迟交付违约金上限 5%）与第十条 10.2（通用违约金 10%）数值不一致。
