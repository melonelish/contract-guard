# P1 样本：办公场地租赁合同（S005）

## 概述

- **合同类型**：租赁合同
- **样本 ID**：S005
- **优先级**：P1（高价值补充验证）
- **页数**：10 页
- **条款数**：10 条（含附件）

## 测试重点

| 焦点 | 说明 |
|------|------|
| 押金退还 | 30个工作日退还期限过长，"场地交接"未定义导致可无限期拖延 |
| 装修权限 | 装修归甲方所有且无补偿，若甲方故意违约解约则可无偿获得装修 |
| 续租条款 | 2.3条与8.2条对续租通知和答复的时间窗口存在冲突 |

## 预期风险分布

| 等级 | 数量 | 关键风险 |
|------|------|----------|
| High | 2 | 押金退还条件模糊、双方解约成本严重不对等 |
| Medium | 3 | 装修无补偿、续租时间冲突、租金年涨幅无上限 |
| Low | 1 | 维修责任划分不够具体 |

## 预期缺失条款

- **物业费与公共事业费用分摊条款**（medium）：费用标准和调整机制未约定
- **保险条款**（medium）：缺少公众责任险和财产险要求
- **场地交还标准条款**（low）：交还的具体标准未定义

## 已知矛盾

第二条 2.3（提前3个月通知续租）与第八条 8.2（甲方30日内答复）形成时间窗口冲突——若甲方最晚答复，乙方仅剩约60天寻找新场地。
